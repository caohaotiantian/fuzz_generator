# 自定义领域知识

本文件包含用于指导 LLM 分析的领域特定知识。

## 协议类型

此示例代码模拟简单的文本协议处理器。

## 输入特征

### buffer 参数
- 类型：字符指针，指向原始输入数据
- 来源：通常来自网络 socket 或文件读取
- 特性：可能包含任意字节序列

### length 参数
- 类型：32位有符号整数
- 有效范围：1 到 255
- 语义：指示 buffer 中有效数据的字节数

## 输出特征

### 返回值
- 0：处理成功
- -1：处理失败（参数无效）

## 数据建模建议

1. buffer 应建模为变长 Blob 类型
2. length 应建模为 Number 类型，范围 1-255
3. 可考虑添加边界条件测试（length=0, length=256）

## 安全注意事项

- strncpy 调用依赖 length 参数，需要特别关注
- 本地缓冲区大小为 256 字节
